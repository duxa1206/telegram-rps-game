# 🎮 Камень, Ножницы, Бумага для Telegram

Простая игра для Telegram Mini Apps **без сервера**!

## 📋 Требования

- Python 3.8+ (только для бота)
- Токен бота от @BotFather
- Аккаунт GitHub (для хостинга)

## 🚀 Быстрый старт

### 1. Получите токен бота

1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### 2. Настройте бота

Создайте файл `bot/.env`:
```
BOT_TOKEN=ваш_токен_здесь
```

### 3. Разместите веб-приложение (БЕСПЛАТНО)

#### Вариант A: GitHub Pages (рекомендуется)

1. Создайте репозиторий на GitHub
2. Загрузите файлы: `index.html`, `style.css`, `script.js`
3. Включите GitHub Pages в настройках репозитория
4. Получите URL вида: `https://ваш-username.github.io/репозиторий/`

#### Вариант B: Vercel

1. Зайдите на https://vercel.com
2. Подключите GitHub аккаунт
3. Импортируйте файлы
4. Получите URL вида: `https://ваш-проект.vercel.app`

### 4. Обновите URL в боте

В `bot/bot.py` замените:
```python
WEBAPP_URL = "https://ваш-username.github.io/репозиторий/"
```

### 5. Запустите бота

```bash
cd bot
pip install -r requirements.txt
python bot.py
```

### 6. Настройте Menu Button

1. Откройте @BotFather
2. Отправьте `/mybots`
3. Выберите вашего бота
4. Bot Settings → Menu Button → Configure
5. Отправьте URL вашего веб-приложения

## 🎮 Как играть

1. Откройте бота в Telegram
2. Нажмите `/start` или кнопку меню
3. Нажмите «🎮 Играть!»
4. Выберите камень, ножницы или бумагу

## 🔥 Преимущества

- ✅ **Нет сервера** — вся логика в браузере
- ✅ **Бесплатный хостинг** — GitHub Pages/Vercel
- ✅ **HTTPS из коробки** — не нужен ngrok
- ✅ **Быстро работает** — нет задержек на backend

## 🔒 Безопасность

- ✅ Токен хранится в `.env` (не в коде)
- ✅ `.env` добавлен в `.gitignore`
- ✅ Никогда не коммитьте `.env` в репозиторий!
