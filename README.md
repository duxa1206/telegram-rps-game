# 🎮 Камень, Ножницы, Бумага

Игра для Telegram Mini Apps.

## Как играть

1. Откройте бота в Telegram
2. Нажмите `/start` или кнопку меню
3. Нажмите «🎮 Играть!»
4. Выберите камень, ножницы или бумагу

## Правила

- 🪨 Камень бьёт ножницы
- ✂️ Ножницы бьют бумагу
- 📄 Бумага бьёт камень
