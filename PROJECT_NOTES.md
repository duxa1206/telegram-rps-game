# 📝 Заметки о проекте

## Что это?
Игра **«Камень, Ножницы, Бумага»** для Telegram Mini Apps.

## Структура
```
telegram_rps_game/
├── index.html      # Веб-приложение (главная страница)
├── style.css       # Стили
├── script.js       # Логика игры (вся в браузере)
├── bot/
│   ├── bot.py      # Telegram бот (открывает Web App)
│   └── .env        # Токен бота (НЕ КОММИТИТЬ!)
├── .gitignore      # Игнор файлов
├── requirements.txt # Зависимости
└── README.md       # Инструкция
```

## Как работает
1. **Бот** (`bot/bot.py`) — только открывает Web App по кнопке
2. **Веб-приложение** — вся логика в браузере (без сервера!)
3. **Хостинг** — GitHub Pages (бесплатный HTTPS)
